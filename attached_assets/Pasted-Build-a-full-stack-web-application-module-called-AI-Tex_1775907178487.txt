Build a full-stack web application module called “AI Text Overlay Animator Tool”.

This tool is part of a larger streaming platform (Creepy-Zone-AI) and must support real-time animated text overlays for live streaming, images, and video.

========================
CORE PURPOSE
========================
User should be able to:
- Type simple text (e.g., "Starting Soon")
- Select a template
- Instantly get a fully styled + animated text overlay
- Use it live in OBS or export as media

========================
NEW FEATURE: TEMPLATE LIBRARY (IMPORTANT)
========================

Create a built-in system of 100+ PRE-MADE TEMPLATES.

Each template must include:
- Font style
- Animation type
- Color theme
- Glow / shadow effects
- Background style (optional)
- Motion behavior

========================

========================
TEMPLATE ENGINE
========================

- User inputs ONLY text:
  Example: "STREAM STARTING SOON"

- Then system:
  1. Applies selected template
  2. Automatically assigns font
  3. Adds animation
  4. Renders preview instantly

- Must support:
  - One-click apply
  - Random template button ("Surprise Me")
